# Known Limitations

- In-memory history
- In-memory master data
- Manual matching not implemented
- Review integration pending

# Technical Debt

- Historical match persistence is still in-memory only.
- Master data repository remains in-memory and not backed by external ERP/CRM adapters.
- MatchingStage configuration does not yet expose external master data source selection.
- Confidence calculators are simple and may need refinement for partial fuzzy or multi-field signal weighting.
- MatchType manual/review handling is a placeholder without active manual review flow.
- `MatchSet` statistics do not yet include per-strategy breakdown.
